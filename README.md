Power BI Business Insights Dashboard
Overview

This project presents a fully interactive Power BI dashboard designed to transform raw data into meaningful business insights.
It helps stakeholders quickly understand performance metrics and make data-driven decisions.

Project Goals
Analyze sales and profit trends
Identify top-performing regions and categories
Track business growth over time
Build an interactive and user-friendly dashboard
Tech Stack
Power BI Desktop
Power Query (ETL)
DAX (Data Analysis Expressions)
Dataset Description

The dataset contains key business metrics such as:

Sales
Profit
Quantity
Discount
Region
Category
Order Date
Project Workflow
1. Data Extraction
Imported dataset into Power BI
2. Data Transformation (ETL)
Cleaned missing values
Changed data types
Structured columns
3. Data Modeling
Built relationships between tables
Created calculated columns
4. DAX Measures

Key measures used:

Total Sales
Total Profit
Average Discount
Total Quantity
5. Dashboard Development

Created interactive visuals:

KPI Cards
Bar Chart (Sales by Region)
Line Chart (Monthly Trend)
Pie Chart (Category-wise Sales)
Slicers for filtering
Key Insights
Top-performing regions identified
Seasonal sales trends observed
High discount areas affecting profit
Category-wise performance analysis
